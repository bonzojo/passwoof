# PassWoof

A login authorization/authentication mini project utilizing EJS, Node/Express & MongoDB <br>
---
![Example](https://github.com/user-attachments/assets/109f1821-40a5-4ba1-92f0-ea7b395ccb36)
---


Fully functional registration process, with login validation, sanitized input and password hashing using bcrypt.

Upon successful registration and login, lands on a basic dashboard page.

Currently not deployed to save resources. To experiment with the project, please feel free to pull repo. For the database you will need to setup your own MongoDB cluster and local environment variables.

This was a practise project to become more familiar with building secure login / authentication.
